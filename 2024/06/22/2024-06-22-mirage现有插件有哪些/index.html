mirage 现有插件有哪些

By limeng32

发表于 2024-06-22

mirage-lowcode

在插件市场搜索 mirage-lowcode 可以获取，此插件是由阿里 lowcode-engine 打包而成。lowcode-engine 是一个非常强大的低代码设计工具，相关使用教程在网上可以找到很多，在下一篇文章中我们会详细介绍如何用 mirage-lowcode 开发新的页面并打包为新的 vscode 插件。

mirage-swagger

在插件市场搜索 mirage-swagger 可以获取，此插件是使用 mirage 从零开发的插件。它的用处是分析指定位置的 swagger json（可以是本机文件地址或远端网址），对 api 进行管理（通过在当前项目下自动生成的配置文件管理），将选中的 api 自动生成为前端调用此 api 时需要的代码，并提供 “生成 js / ts”、“文件统一 / 分离”等选择。本插件核心算法借鉴自网上大牛的项目。

mirage-flow

在插件市场搜索 mirage-flow 可以获取，此插件是由网上大牛写的组件打包而成。这个插件主要展示了 mirage 对本地文件的读/写能力。
